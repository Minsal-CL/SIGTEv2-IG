Resource Profile: Encounter Inicio LE

Official URL: http://minsal.cl/listaespera/StructureDefinition/EncounterInicioLE Version: 0.1.3
Standards status: Draft Maturity Level: 0 Computable Name: EncounterInicioLE

Encounter Inicio LE recurso que se utiliza para representar el encuentro médico, que representa el encuentro de cuando se solicita la interconsulta.

Usage:

  • Use this Resource Profile: BundleInicio LE
  • Refer to this Resource Profile: ServiceRequest LE
  • Examples for this Resource Profile: Encounter/EjemploEncounter
