Resource Profile: Encounter Atender LE

Official URL: https://interoperabilidad.minsal.cl/fhir/ig/tei/StructureDefinition/EncounterAtenderLE Version: 0.1.5
Standards status: Draft Maturity Level: 0 Computable Name: EncounterAtenderLE

Encounter Atender LE recurso que se utiliza para representar el encuentro médico, cuando se comienza a ejecutar la atención de la interconsulta.

Usage:

  • Use this Resource Profile: Bundle Atender LE
  • Refer to this Resource Profile: MessageHeader LE and QuestionnaireResponse Atender LE
  • Examples for this Resource Profile: Encounter/EjemploEncounterAtender
